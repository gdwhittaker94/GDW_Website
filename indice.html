Proyectos

Página de evento de WWF

Esta página es una réplica de un evento real de WWF, con algunos cambios y modificaciones. Es uno de mis proyectos más intensivos en JavaScript, que incluye un carrusel de imágenes bastante complejo, un menú desplegable al hacer clic y un contador regresivo para el evento.

  • HTML5
  • CSS
  • Diseño Web Reactivo
  • Javascript (ES6+)

Sitio web mini de WWF

Este es un proyecto que intentó imitar tres de las páginas del sitio web real de WWF Australia (desde entonces han modificado el diseño original) y fue mi primer sitio web con varias páginas. Fue un proyecto muy intensivo en imágenes, que puso a prueba mis habilidades de CSS y mi capacidad para crear un diseño adaptable. También incluye algunos elementos simples de JavaScript, como un carrusel de imágenes básico y un menú hamburger.

  • HTML5
  • CSS
  • Diseño Web Reactivo
  • Javascript

Testimonios con Flexbox y Grid

Decidí hacer este proyecto de frontendmentor.io para poner a prueba mis habilidades en CSS Flexbox y Grid. Fue un ejercicio útil para comprender cómo funcionan ambos sistemas de diseño, así como las diferencias entre ellos.

  • HTML5
  • CSS
  • Flexbox
  • Grid

Sobre Mí

Antes del verano de 2022, nunca había escrito una línea de código en mi vida 😳 Desde 2018 y hasta ese momento, trabajaba como profesor de inglés, la mayor parte del tiempo en Burgos, en el norte de España - ¡el porqué es una historia interesante pero larga!
Me gusta España 🇪🇸 Hace tiempo que sé que este es el lugar donde quiero estar. Así que enseñar inglés parecía la forma ideal de establecerme económicamente aquí, además de hacer un trabajo que encuentro gratificante y significativo.
¡Era un buen profesor! Sin embargo, después de un par de años en el trabajo, me di cuenta de que estaba listo para asumir algo nuevo y cambiar mi carrera en una nueva dirección 💪
Fue entonces, por pura casualidad, que descubrí la programación 👨‍💻 Al principio no estaba seguro al respecto: "¿Yo, un programador?" - pero cuanto más aprendía al respecto, más me interesaba y quería saber más. Todavía recuerdo la emoción que sentí la primera vez que escribí "Hola mundo" en un editor de texto básico y lo vi aparecer como por arte de magia en una ventana vacía del navegador, ¡woah! 🤩
He progresado mucho desde entonces, ¡y aún sé que hay mucho más por aprender! No ha sido fácil en ocasiones, especialmente al ser autodidacta, sin embargo, aquí sigo hoy aprendiendo y construyendo porque realmente encuentro todo esto de desarrollo web divertido, interesante y gratificantemente desafiante 😃
Ahora, todo lo que quiero hacer es mejorar cada vez más y construir sitios web más hermosos y funcionales para personas y negocios como ustedes. Si les gusta la idea y quieren trabajar conmigo, ¡no duden en ponerse en contacto! ⬇️
