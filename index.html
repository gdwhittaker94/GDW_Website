WWF Mini-Website

This is a project which attempted to mimic three of the pages from the real WWF Australia website (they have since modified the original design) and my first truly multiple-webpage website. It was a very image intensive project, testing my CSS skills and my ability to create a responsive design. There are also some simple JS elements such as a basic image carousel and a hamburger menu.
